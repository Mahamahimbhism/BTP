Welcome to the Stroop Color-Word Test

Instructions:

This test consists of three blocks:

Block 1: Word Reading

You will see color words printed in black ink.

Task: Read the word aloud as quickly as possible.

Example: If you see RED, say "RED"

Block 2: Color Naming

You will see colored squares.

Task: Name the color of each square as quickly as possible.

Example: If you see ■■■, say "RED"

Block 3: Interference (Stroop Effect)

You will see color words printed in mismatched ink colors.

Task: Name the INK COLOR, not the word.

Example: If you see RED, say "BLUE"

⚠️ Important: This test requires you to speak your responses aloud. Please ensure you are in a quiet environment. Press the corresponding key after you've spoken your answer.

Response Keys:

  • R = RED
  • Y = YELLOW
  • G = GREEN
  • B = BLUE
